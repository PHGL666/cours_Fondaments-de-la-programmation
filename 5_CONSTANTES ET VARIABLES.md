# CONSTANTE
valeur qui n'évoluera pas ou très peu

# VARIABLE
valeur qui évoluera très fréquemment


> CONSTANTE et VARIABLE sont des contenants qui contiennent des VALEURS

## CREATION

MOT CLE >>> NOM >>> TYPE >>> VALEUR
var
const
